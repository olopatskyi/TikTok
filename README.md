# TikTok
This is clone of TikTok server application.

There are a few microservices that simulate TikTok
Services:

- Identity:
    - Microsoft Identity
    - ASP.NET Core (.NET 7)
    - SQL Server
    - EntityFrameworkCore 7.x.x

- Chat:
    - Node.js (Express)
    - MongoDB
    - Mongoose
    - Passport.js
    - Socket.IO

- Recomendation:
    - Python (FastAPI)
    - TensorFlow
    - Pandas
    - MongoDB
    - Pymongo


- Notification:
    - Node.js (Express)
    - Postgre
    - Sequelize
    - Passport.js
    - WebSocket

- Media Storage:
    - ASP.NET Core (.NET 7)
    - SQL Server
    - Dapper
    - Google Cloud Storage
    - Content Delivery Network (CDN)

- API Gateway:
    - ASP.NET Core (.NET 7)
    - Ocelot
    - Ocelot Cache Manager
